# Módulo 3: JavaScript desde Cero

---
En este repositorio se encontrarán los ejemplos y ejercicios desarrollados dentro del Módulo 3: JavaScript desde Cero.

---

## 🔴 Objetivo del repositorio
El objetivo de este proyecto es permitir que los estudiantes practiquen conceptos básicos de JavaScript, y luego puedan aplicar las nuevas características de ECMAScript (ES6 y posteriores) para mejorar el código, tal como se describió en los ejercicios previos.

---

## Temas vistos:
- 1. Introducción a JavaScript
    - 1.1. Introducción a JavaScript y su Importancia.
    - 1.2. Tipos de Datos en JavaScript.
    - 1.3. Buenas Prácticas y Comentarios en JavaScript.
- 2. Comparaciones y Decisiones en JavaScript.
    - 2.1. Guardando Valores en Variables y Operadores Básicos.
    - 2.2. Operadores de Comparación y Lógicos.
    - 2.3. Estructura de Control (if).
    - 2.4. Práctica: Comparaciones y Decisiones en JavaScript.
- 3. Arreglos y Ciclos en JavaScript.
    - 3.1. Ciclo 'while' en JavaScript.
    - 3.2. Ciclo 'for' en JavaScript.
    - 3.3. Arrays (arreglos) en JavaScript.
    - 3.4. Práctica: Arreglos y Ciclos.
- 4. Funciones en JavaScript.
    - 4.1. Funciones en JavaScript.
    - 4.2. Práctica: Funciones en JavaScript.
- 5. Intro a Objects en JavaScript.
    - 5.1. Introducción a Objetos en JavaScript.
    - 5.2. Práctica: Objetos en JavaScript.
- 6. Intro al DOM en JavaScript.
    - 6.1. Introducción al DOM con JavaScript.
    - 6.2. Práctica: Intro al DOM en JavaScript.
- 7. Dominando el DOM en JavaScript.
    - 7.1. Acceso y Modificación de Contenido del DOM.
    - 7.2. Acceder a Clases CSS.
    - 7.3. Práctica: Generador de Passwords.
- 8. Proyecto Final (DOM)
 - 8.1. El LocalStorage (Tema Bonus)
 - 8.2. Práctica: App de Comentarios.

## Instrucciones para entender los apuntes:
* Cada tema se compone de diversos archivos. 
* Se dividen en 2 tipos de archivos: El primer tipo de archivos tiene por nombre 'Apuntes_clase.js' donde se registran los subtemas vistos en clases. 
* El otro tipo de archivo contiene el nombre del tema donde se realizan las prácticas vistas en el Campus de DEV.F.
